# Sorting Algorithms Performance Comparison in C++

## Project Overview
This project demonstrates the implementation of five different sorting algorithms in C++ and compares their execution time.

Implemented algorithms:
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

The program accepts user input, sorts the data using each algorithm, and measures execution time using the C++ Chrono library.

---

## Features
- Takes array input from user
- Implements 5 sorting algorithms
- Displays sorted result for each algorithm
- Compares execution time in nanoseconds
- Helps analyze algorithm efficiency

---

## Technologies Used
- C++
- Standard Template Library (STL)
- Chrono Library (`<chrono>`)

---

## Sorting Algorithms Included

### 1. Bubble Sort
Repeatedly compares adjacent elements and swaps them if they are in the wrong order.

**Time Complexity:**  
- Best: O(n)  
- Average: O(n²)  
- Worst: O(n²)

---

### 2. Selection Sort
Finds the smallest element and places it at the correct position.

**Time Complexity:**  
- Best: O(n²)  
- Average: O(n²)  
- Worst: O(n²)

---

### 3. Insertion Sort
Builds sorted array one element at a time by inserting elements in proper position.

**Time Complexity:**  
- Best: O(n)  
- Average: O(n²)  
- Worst: O(n²)

---

### 4. Merge Sort
Uses Divide and Conquer by splitting array into halves and merging sorted parts.

**Time Complexity:**  
- Best: O(n log n)  
- Average: O(n log n)  
- Worst: O(n log n)

---

### 5. Quick Sort
Selects a pivot and partitions array around it.

**Time Complexity:**  
- Best: O(n log n)  
- Average: O(n log n)  
- Worst: O(n²)
